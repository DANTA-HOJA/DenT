# Instruction

## 提交 Job
``` bash
sbatch <FILE_NAME>.sh # 附註： 提交完成後，將顯示系統派發的 Job ID。
```

## 監看輸出檔
```bash
tail -f slurm-<JOB_ID>.out
```
